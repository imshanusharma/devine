# Devine  
## React-Based Social Media App  

Devine is a social media app, designed to connect developers.  

- React  
- Node Js  
- Mongo DB  
- Express  

## Features  

- Creation and deletion of Post.  
- Follow / Unfollow friends.  
- Like and Dislike a post.  
- Login / Logout.  
- Searching profiles.
